# 智能食物记录 - 设计风格提案

## 设计理念

**极简主义 + 食物温度感**

本产品的核心设计理念是"零输入负担"，因此视觉设计应该：
- 减少认知负担，突出关键数据
- 使用温暖的色彩传达食物的温度感
- 保持界面清爽，避免信息过载

---

## 方案一：「清新健康风」

### 色彩系统

```
主色调：
- 品牌蓝 #3498db（主按钮、链接、强调）
- 清新绿 #2ecc71（健康、达标、缺口）
- 警示橙 #f39c12（警告提示）

辅助色：
- 背景灰 #f5f7fa（页面背景）
- 卡片白 #ffffff（卡片背景）
- 文字黑 #2c3e50（主标题）
- 描述灰 #7f8c8d（辅助文字）

功能色：
- 成功 #2ecc71
- 警告 #f39c12
- 危险 #e74c3c
```

### 字体系统

```
标题：PingFang SC / Helvetica Neue
- 大标题 1.5rem (24px) bold
- 卡片标题 1.3rem (21px) bold
- 小标题 1.1rem (18px) medium

正文：PingFang SC / SF Pro Text
- 正文 1rem (16px) regular
- 辅助文字 0.9rem (14px) regular
- 小字 0.8rem (13px) regular
```

### 圆角系统

```
按钮：50px（完全圆角）
卡片：20px（大圆角）
输入框：10px（中圆角）
小元素：4-8px（小圆角）
```

## 推荐方案

**推荐使用「方案一：清新健康风」**

**理由：**
1. 色彩心理学：蓝绿色传达健康、科学、可信的感觉
2. 认知负荷：灰白背景减少视觉干扰，突出核心数据
3. 行业惯例：符合健康/饮食类应用的主流审美
4. 开发成本：颜色和样式实现简单，维护成本低
5. 可扩展性：未来添加新功能易于融入现有设计语言

## 组件规范

### 按钮

```
主要按钮 (btn-primary)
- 背景: #3498db
- 文字: 白色
- 高度: 44-50px
- 圆角: 15px
- 阴影: 0 4px 20px rgba(52, 152, 219, 0.3)

次要按钮 (btn-secondary)
- 背景: 白色
- 文字: #7f8c8d
- 边框: 2px solid #ecf0f1
- 其他同上

链接按钮 (btn-link)
- 背景: 透明
- 文字: #7f8c8d + 下划线
```

### 卡片

```
标准卡片
- 背景: 白色
- 圆角: 20px
- 内边距: 20-30px
- 阴影: 0 4px 20px rgba(0, 0, 0, 0.08)
```

### 输入框

```
表单输入
- 背景: 白色
- 边框: 2px solid #ecf0f1
- 圆角: 10px
- 内边距: 12px 15px
- focus时: 边框变蓝 #3498db
```

### 图标

```
Emoji图标
- 大图标: 2.5rem (40px)
- 中图标: 2rem (32px)
- 小图标: 1.2rem (19px)
```

## 响应式适配

```
断点定义：
- 手机: < 480px
- 平板: 480px - 768px
- 桌面: > 768px

本应用主要针对移动端设计，
最大宽度限制: 400px (居中显示)
```

---

## 动画规范

```
标准过渡: transition: all 0.2s
慢速过渡: transition: all 0.3s ease
进度条: transition: width 0.3s ease
模态弹窗: animation: modalSlideIn 0.3s ease
```

---

## 可访问性

```
对比度要求：
- 正文文字: 至少 4.5:1
- 大文字(18px+): 至少 3:1
- 图标/图形: 至少 3:1

触摸目标：
- 最小点击区域: 44x44px
```
